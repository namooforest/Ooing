# Ooing - 카드명세서 → 후잉 업로더

카드사 이메일 명세서를 브라우저에서 열고, 거래 내역을 자동으로 읽어서 [후잉(Whooing)](https://whooing.com) 가계부에 업로드하는 크롬 확장 프로그램입니다.

## 지원 카드사

| 카드사 | 파서 버전 | 상태 |
|--------|-----------|------|
| 삼성카드 | v1.0 | 지원 |
| 현대카드 | v1.0 | 지원 |
| 기타 | - | 미지원 |

## 주요 기능

- 이메일 명세서(보안메일) 자동 감지 및 파싱
- 삼성카드 "더보기" 버튼 자동 클릭으로 전체 항목 로드
- 후잉 API 연동 (API Key / OAuth 2.0)
- 아이템명 자동완성 (후잉 최근 입력 기록 기반)
- 아이템명 → 계정 자동 매핑 (이전 업로드 이력 학습)
- 계정 기간(open_date ~ close_date) 기반 필터링
- 10건 단위 페이지 처리 및 일괄/개별 계정 선택
- 아이템명·금액 직접 수정, 항목 삭제

## 설치 방법

1. 크롬 브라우저에서 `chrome://extensions`로 이동
2. **개발자 모드** 활성화
3. **압축해제된 확장 프로그램을 로드합니다** → 이 폴더 선택

## 초기 설정

1. 확장 프로그램 아이콘 클릭 → 우측 상단 ⚙️ 설정 버튼
2. 후잉 API Key 입력 (후잉 > 계정 > 비밀번호 및 보안 > AI 연동)
3. **연결 테스트**로 확인
4. **설정 저장**

## 사용 방법

### 현대카드
1. 현대카드 이메일 명세서(HTML)를 브라우저에서 열기
2. 보안메일 비밀번호(생년월일 6자리) 입력하여 복호화
3. 복호화된 명세서가 표시되면 확장 프로그램 아이콘 클릭
4. **현재 페이지에서 명세서 읽기** 클릭
5. 현대카드 자동 감지 → JavaScript 내 `UseDesc` 데이터에서 거래 내역 직접 추출

### 삼성카드
1. 삼성카드 이메일 명세서(HTML)를 브라우저에서 열기
2. 보안메일 비밀번호 입력 → `bill.samsungcard.com`으로 자동 리다이렉트
3. 명세서 웹페이지가 로드된 후 확장 프로그램 아이콘 클릭
4. **현재 페이지에서 명세서 읽기** 클릭
5. 삼성카드 자동 감지 → "더보기" 자동 클릭 후 전체 거래 내역 추출

### 공통
- 후잉 섹션 선택 후, 각 거래 항목별로 왼쪽(차변)/오른쪽(대변) 계정을 선택
- 상단 일괄 계정 선택으로 전체 항목 한번에 변경 가능
- 이전에 동일 아이템명으로 업로드한 이력이 있으면 계정이 자동 매핑됨
- 10건씩 페이지 처리되며, **업로드(n/m)** 버튼으로 현재 페이지의 항목을 업로드
- 업로드 완료된 항목은 목록에서 자동 제거

## 프로젝트 구조

```
Ooing/
├── manifest.json              # 확장 프로그램 매니페스트
├── background.js              # 서비스 워커 (팝업 윈도우 관리)
├── popup/
│   ├── popup.html
│   ├── popup.css
│   └── popup.js               # 메인 UI 로직
├── options/
│   ├── options.html
│   ├── options.css
│   └── options.js             # API 인증 설정
├── lib/
│   ├── whooing-api.js         # 후잉 API 클라이언트
│   ├── card-parser.js         # 공통 파싱 유틸 (날짜, 금액)
│   ├── page-scraper.js        # 페이지 DOM 추출 (탭에 주입)
│   └── parsers/
│       ├── index.js           # 카드사 자동 감지 + 파서 라우터
│       ├── samsung-v1.js      # 삼성카드 파서 v1
│       └── hyundai-v1.js      # 현대카드 파서 v1
└── icons/
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```

## 카드사 파서 추가 방법

새 카드사를 추가하려면:

1. `lib/parsers/` 아래에 `새카드사-v1.js` 파일 생성
2. `VERSION` 상수와 `parse(scrapedData)` 함수 구현
3. `CardParsers.registerParser('카드사id', 파서객체)` 호출
4. `lib/parsers/index.js`의 `SUPPORTED_CARDS`에 카드사 등록
5. `popup/popup.html`에서 스크립트 추가
6. `manifest.json`의 `web_accessible_resources`에 추가
